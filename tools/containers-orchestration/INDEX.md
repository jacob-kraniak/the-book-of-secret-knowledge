# Containers & Orchestration

Collection of CLI tools, web-based management platforms, reverse proxies/load balancers, security utilities, and related resources for container runtimes (Docker, containerd, etc.) and orchestration systems (primarily Kubernetes).

Focuses on actionable tools following the [TOOL-TABLE-SCHEMA](/TOOL-TABLE-SCHEMA.md). Learning materials, cheat sheets, tutorials, best practices, and failure stories are housed in [/documentation/containers-orchestration/](/documentation/containers-orchestration/INDEX.md).

## Subcategories

- [CLI Tools](./cli-tools.md) — command-line utilities for container inspection, metrics, runtime sandboxes, and monitoring
- [Web / Management Tools](./web-tools.md) — dashboards, API gateways, reverse proxies, load balancers, and full container/Kubernetes management platforms
- [Security](./security.md) — vulnerability scanners, hardening benchmarks, secure registries, signing tools, and offensive/pentest container images
- [Learning & Best Practices](/documentation/containers-orchestration/INDEX.md) — cheat sheets, awesome lists, tutorials, labs, production guides, security checklists, and failure stories
